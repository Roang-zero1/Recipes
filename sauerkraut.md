# Sauerkraut

# Ingredients

|          | Ingredient                  |
| -------- | --------------------------- |
| 4 Körner | Pfeffer                     |
| 4 Beeren | Wacholder                   |
| 1 Blatt  | Lorbeer                     |
| 1TL      | Suppenpulver                |
| 1TL      | Bratensaft (wenn verfügbar) |

## Directions

1. Sauerkraut aufkochen
1. Zutaten hinzufügen
