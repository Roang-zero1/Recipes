# Mehlknödel

## General

|        |         |
| ------ | ------- |
| Prep   | 15 mins |
| Cook   | 1 hr    |
| Serves | 4       |

## Ingredients

|       | Ingredient         |
| ----- | ------------------ |
| 60dag | mehlige Kartoffeln |
| 30dag | glattes Mehl       |
| 1TL   | Salz               |

## Directions

1. Kartoffeln kochen
1. Kartoffeln schälen
1. Mehl auf der Arbeitsplatte ausbreiten
1. Kartoffeln in das Mehl pressen und Knödel formen
1. Knödel 12min kochen, schwimmen evtl. nicht auf
